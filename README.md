# 🚀 Skill Gap Analysis & Career Recommendation Tool

A web-based platform that analyzes user skills and recommends suitable career paths with a learning roadmap and interview preparation.

This project helps users understand their **skill gaps** and provides guidance to reach their desired career.

---

## 📌 Features

- 🔍 Skill Gap Analysis  
- 🎯 Career Path Recommendation  
- 📊 Career Match Score  
- 🛤 Personalized Learning Roadmap  
- 🎤 Mock Interview Preparation  
- 🏆 Achievement Badges and Progress Tracking  

---

## 🛠 Technologies Used

**Frontend**
- React
- Vite
- JavaScript
- HTML5
- CSS3

**Backend / Platform**
- Base44 (for data entities and application backend)

## ⚙️ Installation & Running the Project

1. Clone the repository

git clone https://github.com/YOUR\_USERNAME/career-path-ai.git

2. Navigate to project folder

cd career-path-ai

3. Install dependencies

npm install

4. Run the development server

npm run dev

5. Open in browser

http://localhost:5173

## 🌐 Application Demo

Click **Launch Application** on the landing page to open the full Career Path AI system.

## 💡 Use Case

This platform helps:

- Students identify career opportunities
- Professionals analyze skill gaps
- Users prepare for technical interviews
- Learners follow structured career roadmaps

---
## System Architecture

User → Frontend (React + Vite) → Base44 Backend → Data Entities

1. User inputs skills and career interests
2. System analyzes skill gaps
3. Career paths and learning roadmap are generated
4. Users track progress and prepare for interviews
